Tic-tac-toe

A simple tic-tac-toe game created with TypeScript, HTML, CSS. It is a small personal project for TypeScript practicing.

Description

Tic-tac-toe is a simple two-player game. The two players turn to mark the 3 x 3 grid with X or O. This game always starts with player X drawing first. The player who draws three of their marks in a line (in horizontal, vertical, or diagonal) wins. The game results in a tie if there is no winner when all places in the grid are taken.
